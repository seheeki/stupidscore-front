## Node.js 설치

[https://nodejs.org/en/](https://nodejs.org/en/)

## Vue CLI 설치

```
npm install -g @vue/cli
```

## Vue 프로젝트 생성

```
vue create mvvm-01
```
* Vue.js 3.x 버전 선택
* Router 선택

## axios 설치 setup

```
cd mvvm-01
npm install axios
```

## Compiles and hot-reloads for development

```
npm run serve
```
